# iSoftrend Billing System

## Setup
```bash
pip install flask reportlab Pillow
python app.py
```
Then open http://localhost:5000

## Features
- Quotation → Proforma Invoice → Tax Invoice workflow
- Customer management (Individual / Business with GST)
- Item/Service catalog
- PDF generation matching iSoftrend template
- Changeable signature from Settings
- Data stored in JSON files
